Login-register-app-using-crud-backend

This repository contains the backend API for a Login and Registration application, handling user authentication, registration, and CRUD (Create, Read, Update, Delete) operations.

The project is built using ASP.NET Core Web API and follows a layered architecture to ensure clean code, scalability, and maintainability.
